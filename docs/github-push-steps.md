# GitHub Push Steps from VS Code

## 1. Create a new GitHub repo

Go to GitHub and create a repository named:

```text
ai-workflow-automation-platform
```

Keep it public. Do not add a README because this project already includes one.

## 2. Open this folder in VS Code

```text
File -> Open Folder -> ai-workflow-automation-platform
```

## 3. Open terminal in VS Code

```text
Terminal -> New Terminal
```

## 4. Run these commands

```bash
git init
git add .
git commit -m "Initial commit: AI workflow automation platform"
git branch -M main
git remote add origin https://github.com/Poojitha-g15/ai-workflow-automation-platform.git
git push -u origin main
```

## 5. Add repo description

```text
Full-stack AI-assisted workflow automation platform using Python, React, FastAPI, PostgreSQL, REST APIs, OpenAI/Claude service layer, Docker, and AWS-ready deployment.
```
