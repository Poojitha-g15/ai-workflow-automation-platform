# Project Notes

This project was created from the resume concept:

**AI-Assisted Workflow Automation Platform — Python, React, Node.js, REST APIs, OpenAI/Claude, PostgreSQL, AWS**

The goal is to show recruiter-friendly, practical engineering skills:

- Full-stack development
- Backend API design
- Prompt engineering
- AI service integration
- Database persistence
- Evaluation of generated output
- Dockerized local development
- AWS-ready deployment thinking

## What to customize before applying

1. Add screenshots to `docs/screenshots/`.
2. Push the project to GitHub.
3. Add the GitHub link to your resume.
4. Mention one real improvement, such as response consistency score or workflow classification accuracy, after you test the app with sample requests.
